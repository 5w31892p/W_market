## 주요 기능

#### USER
- 회원가입
- 로그인
- 닉네임변경
- 회원탈퇴

#### PRODUCT
- 상품등록
- 전체 상품 리스트
- 상품 상세보기
- 상품 수정
- 상품 거래상태 수정
- 상품 삭제

#### CHAT
- 상품에 대한 1:1 채팅방 생성
- 나의 채팅방 리스트 보기
- 특정 채팅방 가져오기
- 채팅방 삭제하기
- Kafka를 사용한 실시간 채팅 기능
